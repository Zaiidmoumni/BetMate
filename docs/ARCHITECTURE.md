# BetMate Architecture

## High-Level Overview

```
Client Layer (React) → API Layer (NestJS) → Data Layer (MongoDB)
```

## Tech Stack

### Frontend
- React 18 + TypeScript + Vite
- Redux Toolkit, React Router v6
- Radix UI + Tailwind CSS

### Backend
- NestJS 10 + Node.js 20
- MongoDB + Mongoose
- JWT Auth + Socket.IO

## Modules

- auth, users, events, bet, payment, odds-api, mail
