# BetMate - Monorepo

A full-stack betting platform with NestJS (API) and React (Web).

## Structure

```
├── api/          # NestJS backend
├── web/          # React frontend
├── packages/     # Shared packages
├── CONTRIBUTING.md
├── CODE_OF_CONDUCT.md
└── SECURITY.md
```

## Quick Start

```bash
npm install
npm run dev       # Run both apps
```
